# Excel-Payroll-Basics 

This repository contains my Excel Payroll project from a section of a FreeCodeCamp Excel course. The project progressed through three assignments, each building on the previous, culminating in the final payroll PDF. 

> **Course link:** [FreeCodeCamp Excel Course](https://youtu.be/Vl0H-qTclOg?si=32HITWjHd63anquc)  
> **Note:** All data was created by the instructor during the lessons. All formulas, calculations, and formatting in this repository were completed independently by me.

---

## Project Overview

The project demonstrates practical Excel skills applied in stages:

1. **Assignment 1: Basic Formulas**  
   **Skills gained:** Arithmetic operations, SUM, AVERAGE, MIN, MAX functions

2. **Assignment 2: Conditional Logic**  
   **Skills gained:** IF statements, decision-making based on conditions

3. **Assignment 3: Cell Referencing & Formatting**  
   **Skills gained:** Relative and absolute cell references, cell coloring, exporting to PDF

4. **Payroll: Final Consolidated Project**
